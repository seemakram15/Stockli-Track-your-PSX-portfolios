# Stockli

Stockli is a PSX-focused investing platform built to help users track portfolios, monitor market activity, review company fundamentals, compare stocks, and manage alerts in one place.

## Services

- Portfolio tracking and performance monitoring
- Profit and loss insights
- Market watch and sector monitoring
- Stock fundamentals and financial analysis
- Peer comparison across listed companies
- Watchlists and price alerts
- Calendar-based portfolio activity views
- Mobile-friendly web experience

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Supabase
- Redis
- Progressive Web App support
